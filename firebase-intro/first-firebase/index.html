Google Firebase Startup Steps

  1. Visit Google Firebase

  2. Click on Get Started

  3. Click on Add Project

  4. Input a project name and whether or not you want to enable Google Analytics

  5. Click on to add firebase to your application

  6. Create an app nickname and register your app

  7. Copy the code that they give you there and paste it somehwere

  8. Go back to the App Home and click on Realtime Database

  9. Click on "Create Database"

  10. Enable the database with "Start in lock mode"
